```
冒泡算法(maopao sort)：从小到大排序
原理：1轮中，将最大的数字放到最后。从第一个和第二个数字开始，每次比较两个数字，将大的放到后面，
依次遍历，直到最大的数字放到最后。
N轮中，将前面最大的数字放到第length-N的位置
时间复杂度：o（n2）
空间复杂度：o（1）
稳定性：稳定
```

```
选择排序（Selection Sort）：从小到大
原理：i轮中，第i个位置为需要交换数字，找出i+后面的所有最小数字（k处），
判断是否需要交换i和k
N-1轮后，则完成
时间复杂度：o（n2）
空间复杂度：o（1）
稳定性：不稳定
```

```
插入排序（Insertion Sort）：从小到大
原理：i轮中，前面i-1个数字都是有序的，第i个需要和前面i-1个数字对比，
并将i的数字插入其中，如：i-3的数字比i的数字要大，i-4的数字比i小，
则需要挪动成i，i-3，i-2，i-1
N轮后，则完成
时间复杂度：o（n2）
空间复杂度：o（1）
稳定性：稳定
```

```
希尔排序（Shell Sort）：从小到大
原理：类似插入算法
i轮：间隔gap为5的话，表示0和5的数字判断是否交换，1和6的数字判断是否交换
i+1轮：间隔gap为3的话，表示0和3的数字判断是否交换，1和4的数字判断是否交换
N轮：间隔为0的话，表示做一次完整的插入排序算法


```

```
归并排序：merge sort：从小到大
原理：将一个大的数组拆分成多个小的数组，然后两两合并，且合并后的数组是从小到大的有序的
```
