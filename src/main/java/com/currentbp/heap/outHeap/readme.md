
###堆外内存
博客：https://www.jianshu.com/p/17e72bb01bf1

引用pom
~~~
        <dependency>
            <groupId>org.ehcache</groupId>
            <artifactId>ehcache</artifactId>
            <version>3.4.0</version>
        </dependency>
~~~

1. HelloHeapTest:这个类是使用ehcache包的堆外内存使用方法
2. HelloHeapUnsafeTest:这个类是使用unsafe的堆外内存使用方法
